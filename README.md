# Site Web La Qualia

Site vitrine statique pour La Qualia, entreprise située au Saguenay et spécialisée en intelligence artificielle, automatisation, transformation numérique, formation IA, ROI, KPI et accompagnement IA.

## Contenu du livrable

- `index.html` : page d'accueil complète avec SEO, Open Graph et schema.org.
- `styles.css` : design responsive premium, sombre, chaleureux et aligné sur la charte visuelle.
- `script.js` : menu mobile, animation légère au défilement et formulaire mailto.
- `assets/` : logo SVG, favicon SVG, visuel héro et image Open Graph.
- `robots.txt` : directives d'indexation.
- `sitemap.xml` : sitemap simple.
- `_headers` : en-têtes de sécurité pour Cloudflare Pages.

## Déploiement rapide

### Cloudflare Pages

1. Créer un projet Cloudflare Pages.
2. Connecter le dépôt GitHub contenant ces fichiers ou téléverser le dossier.
3. Framework preset : `None`.
4. Build command : laisser vide.
5. Output directory : `/`.
6. Déployer.

### Netlify

1. Glisser-déposer le dossier `la-qualia-site` dans Netlify.
2. Aucun build requis.
3. Associer le domaine personnalisé.

### GitHub Pages

1. Déposer les fichiers à la racine du dépôt.
2. Activer GitHub Pages dans les paramètres.
3. Choisir la branche et le dossier racine.

## À personnaliser avant publication

- Remplacer `https://laqualia.com/` dans `index.html`, `robots.txt` et `sitemap.xml` si le domaine final est différent.
- Ajouter un numéro de téléphone seulement si souhaité.
- Brancher le formulaire à un service externe comme Formspark, Formspree, Make, Zapier ou un formulaire natif si l'envoi par courriel n'est pas suffisant.
- Ajouter Google Search Console et un outil d'analyse si désiré.

## Positionnement intégré

La page met en valeur les cinq services de La Qualia : diagnostic IA, préparation de l'entreprise, solutions IA personnalisées, automatisation intelligente et Académie. Elle présente aussi les trois trajectoires d'accompagnement, la méthode en six étapes, les notions de ROI et KPI, ainsi que la sécurité et l'utilisation responsable de l'IA.
